## Refine проблемы:
 - Накопление ошибок → каждая итерация может искажать предыдущую summary
 - Потеря временного контекста → "кто когда что сказал"
 - Дрейф фокуса → итоговая summary может уйти от первоначальных ключевых моментов

## Map-Reduce проблемы:
 - Локальный контекст → каждый чанк обрабатывается изолированно
 - Агрегация теряет нюансы → финальный reduce не видит тонкостей
 - Проблема границ чанков → важные связи разрываются между чанками

 ## Улучшения:
  - Семантический чанкинг
  - Добавление векторной/графовой памяти
